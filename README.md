## What's this?

scheme-d is an incomplete Scheme interpreter in D for an easy embeddable language.


## Licenses

## Usage

See: https://github.com/p0nce/scheme-d/blob/master/examples/repl/repl.d

## Currently supported

- special forms
- UTF8 string literals
- double literals
- + - * /
- custom builtin functions
